# E-Commerce Website

A fully responsive and modern e-commerce web application front-end built entirely with HTML, CSS, and Vanilla JavaScript. This project focuses on delivering a clean, aesthetic user experience alongside functional, dynamic features, completely independent of external CSS frameworks.

## Key Features

* **Dynamic Product Catalog:** Advanced filtering (by category, brand, price range, rating) and sorting capabilities using a mock JavaScript database.
* **Interactive Shopping Cart:** Real-time cart updates, quantity management, and a "Save for Later" feature, all persisted using browser `localStorage`.
* **Modern UI/UX:** A clean layout with responsive design for both mobile and desktop, including a mobile sidebar navigation, grid/list view toggles, and interactive hover states.
* **Detailed Product Pages:** Features image galleries, dynamic pricing tiers based on quantity, and tabbed product information.
* **Interactive Components:** Includes custom interactive elements like a deals countdown timer and animated slide-in toast notifications for supplier requests.

## Tech Stack

* **HTML5**
* **CSS3:** Custom styling, CSS variables, Flexbox, and Grid layouts.
* **Vanilla JavaScript:** DOM manipulation, LocalStorage integration, and complex event handling.
